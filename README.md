# Intertec — CMV e Rentabilidade

Sistema de gestão de CMV (Custo da Mercadoria Vendida) e rentabilidade da Intertec.
Calcula custos, margens e resultados financeiros com precisão total (dinheiro nunca em float).
Toda a documentação de requisitos e regras de cálculo está na pasta [docs/](docs/).
As regras de trabalho para desenvolvimento estão em [CLAUDE.md](CLAUDE.md).
